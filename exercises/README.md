# Exercises

This directory contains practice exercises and assignments for the Software Testing course.

## Structure

Each exercise will be organized in its own subdirectory with:
- Source code
- Test cases
- Documentation
- Any required configuration files

## Guidelines

- Follow the naming convention: `exercise-##-description`
- Include a README.md in each exercise directory
- Ensure all tests are properly documented
- Use appropriate testing frameworks for the language used

---

*Exercise materials will be added as the course progresses.*
