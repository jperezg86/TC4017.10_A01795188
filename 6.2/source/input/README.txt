Entradas de prueba para hotel_system

TC1_*_valid.txt
- Contiene registros correctos para hoteles, clientes y reservaciones.
- Sirve para validar carga normal del sistema.

TC2_*_invalid.txt
- Contiene registros con errores de formato y de negocio.
- Sirve para comprobar que el sistema muestre errores en consola
  y continue ejecutando sin detenerse.

Formato sugerido:
- Un registro JSON por linea (estilo JSONL).
