## About This Project

This repository contains the coursework, exercises, and projects for the Software Testing and Quality Assurance course (TC4017.10). The course focuses on modern software testing practices, quality assurance methodologies, and ensuring software reliability.

## Course Topics

- Software Testing Fundamentals
- Unit Testing
- Integration Testing
- System Testing
- Test Automation
- Quality Assurance Best Practices
- Continuous Integration/Continuous Deployment (CI/CD)
- Test-Driven Development (TDD)
- Behavior-Driven Development (BDD)
